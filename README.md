# Full-Stack E-Commerce Interface
A high-performance, responsive e-commerce frontend built with React, designed to interface seamlessly with a custom Express.js backend. This project focuses on efficient state management, asynchronous data orchestration, and a modular architecture to provide a seamless user shopping experience.

# Key Features
Dynamic Data Orchestration: Real-time synchronization of cart, delivery, and payment summaries using React's lifecycle hooks and axios.

Modular Architecture: Clean separation of concerns with dedicated folders for pages, components, and utils.

Responsive Design: Optimized for seamless performance across desktop and mobile devices.

State-Driven UI: UI components automatically react to cart modifications, ensuring the user always sees accurate pricing and order details.

# Tech Stack
Frontend: React, JavaScript, CSS

Communication: Axios (for RESTful API integration)

Build Tools: Vite/npm

Backend Interface: Express.js REST API

# Getting Started
Clone the repo: git clone https://github.com/SadushBeshiri/E-commerce-project.git

Install dependencies: npm install

Run the app: npm run dev
